# Gemini Spark Connector Map

Use this map to decide which Gemini Spark skills need which connected apps.

| Connector | Best Skill Types | Approval Watchouts |
|---|---|---|
| Gmail | Inbox summaries, reply drafts, follow-ups, client intake, receipts | Sending emails, replying, forwarding, archiving, deleting |
| Google Calendar | Daily planning, meeting prep, event drafting, time blocking | Creating, editing, deleting, or moving events |
| Google Drive | File discovery, organization, folder systems, source gathering | Moving, deleting, sharing, or changing permissions |
| Google Docs | SOPs, guides, notes, summaries, reports | Creating or editing documents; sharing docs externally |
| Google Sheets | Dashboards, trackers, logs, reporting | Editing rows, formulas, sharing sheets |
| Google Slides | Deck outlines, slide drafts, presentation plans | Creating or editing decks; sharing externally |
| Google Keep | Idea capture, note cleanup, task extraction | Editing or deleting notes |
| Google Tasks | Task creation, prioritization, reminders | Creating, completing, or deleting tasks |
| YouTube | Video research, content analysis, creator inspiration | Avoid copying copyrighted scripts or misrepresenting source content |
| Google Maps | Itineraries, location research, local planning | Booking, reservations, route commitments |
| Canva | Creative briefs, post concepts, design directions | Publishing, sharing, or modifying live brand files |
| Google Photos | Asset finding, photo organization, visual content libraries | Privacy-sensitive visual searches and sharing photos |
| GitHub | Repo summaries, docs, issue planning, beginner explanations | Code changes, pull requests, issues, secrets, credentials |

## Simple Connector Rule

Connect the fewest apps needed for the workflow.

Example: A Daily Brief may need Calendar, Gmail, Tasks, and Drive. A Canva Creative Brief may only need Docs and Canva.
