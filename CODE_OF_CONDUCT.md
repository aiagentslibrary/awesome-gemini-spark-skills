# Code of Conduct

This project is intended to be helpful, beginner-friendly, and respectful.

Contributors should:

- Keep feedback constructive
- Avoid harassment, spam, or personal attacks
- Respect privacy and safety concerns
- Avoid submitting unsafe automation patterns
- Keep skills clear, practical, and approval-first

Maintainers may remove issues, pull requests, or comments that are abusive, unsafe, spammy, or unrelated to the project.
