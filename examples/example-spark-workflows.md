# Example Gemini Spark Workflows

These examples show how the skills in this repository can be combined into practical Spark workflows.

## 1. Daily Business Brief

**Apps:** Gmail, Calendar, Drive, Tasks  
**Skill:** `spark-daily-brief`

```text
Every weekday morning, review my Gmail, Calendar, recent Drive activity, and open tasks. Create a concise daily business brief with priorities, meetings, urgent emails, follow-ups, and suggested next actions. Do not send emails, change calendar events, or edit files without approval.
```

## 2. Inbox to Task List

**Apps:** Gmail, Tasks, Keep  
**Skill:** `spark-inbox-to-tasks`

```text
Review my unread and important emails from the last 48 hours. Extract action items, group them by urgency, and draft a prioritized task list. Ask before creating or assigning any tasks.
```

## 3. Meeting Prep Workflow

**Apps:** Calendar, Gmail, Docs, Drive  
**Skill:** `spark-meeting-prep`

```text
Before my next meeting, review the calendar event, related emails, and recent documents. Create a meeting prep brief with context, agenda suggestions, open questions, and follow-up items from past conversations. Ask before editing or sharing anything.
```

## 4. Weekly Content Calendar

**Apps:** YouTube, Docs, Sheets, Canva  
**Skill:** `spark-content-engine`

```text
Create a weekly content calendar from my recent ideas, top-performing posts, YouTube research, and saved notes. Organize the ideas into hooks, post formats, captions, and design briefs. Ask before posting or creating public assets.
```

## 5. Client Intake Summary

**Apps:** Gmail, Docs, Sheets, Calendar  
**Skill:** `spark-client-intake`

```text
Review the latest client inquiry emails and organize each lead into a simple intake summary. Include business type, need, urgency, missing details, suggested reply, and next step. Ask before replying or adding calendar events.
```

## 6. Google Drive Cleanup

**Apps:** Drive, Docs, Sheets, Slides  
**Skill:** `spark-drive-organizer`

```text
Review my recent Google Drive files and suggest a cleaner folder structure. Identify duplicates, old drafts, unclear file names, and files that should be grouped together. Do not move, delete, rename, or share files without approval.
```

## 7. Weekly CEO Dashboard

**Apps:** Gmail, Calendar, Sheets, Docs, Tasks  
**Skill:** `spark-weekly-ceo-dashboard`

```text
Every Friday afternoon, create a weekly business dashboard from my emails, meetings, tasks, and key trackers. Summarize wins, blockers, missed follow-ups, revenue/admin items, and priorities for next week. Ask before changing any source files.
```
